MavensMate Salesforce IDE for Sublime Text

Welcome to MavensMate for Sublime Text! Our mission at MavensMate is to make building Salesforce applications easy and fun regardless of platform or text editor.

SUPPORTED OPERATING SYSTEMS:
	- OSX 10.11.x+
	- Windows 7, 8, Server 2008
	- Linux Ubuntu 14

PREREQUISITES:
	- MavensMate Desktop (https://github.com/joeferraro/MavensMate-Desktop) MUST BE INSTALLED AND RUNNING in order for MavensMate for Sublime Text to function properly
